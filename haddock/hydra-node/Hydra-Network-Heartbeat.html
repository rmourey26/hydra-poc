Hydra.Network.Heartbeat

Description

A naive implementation of an application-level Heartbeat This module exposes a Component withHeartbeat than can be used to wrap another NetworkComponent and piggy-back on it to send and propagate HeartbeatMessages.

Its current behavior is very simple: When it starts, it sends a Heartbeat message with its own identifier every 500ms, until the wrapped component sends another message. Heartbeat messages received from other components are simply propagated to the wrapped component.
